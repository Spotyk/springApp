# springApp 
Internet shop, with Admin Panel, user cabinet, orders page and cart. L10N is done.
Used technologies:
  Spring Boot
  PostgreSQL
  Maven
  Freemarker
  JPA
  @Configuration + @Bean
  @Entity
  @Service
  @Controller
  @Repository
  @Component
  Interfaces
  OOP
  JavaScript(JQuery, AJax)
  CSS3 & HTML5
  UML diagrams
  
  It is my diploma and some it could contains of some bugs or not correct hot fixes.
